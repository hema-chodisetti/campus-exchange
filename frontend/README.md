# CampusExchange - Frontend

Angular 21 single-page application for the CampusExchange marketplace.

## Pages
- **Home** - Browse, search, and filter listings
- **Login / Register** - FAU email authentication
- **Listing Detail** - View listing, place bids, message seller
- **Create / Edit Listing** - Manage your listings
- **Messages** - Conversations with buyers/sellers
- **Profile** - View profile and your listings
- **Admin Dashboard** - Review reported listings (admin only)

## Running
This service runs via Docker Compose from the project root. See the main [README](../README.md) for setup instructions.
